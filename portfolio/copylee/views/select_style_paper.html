Custom Select box jquery plugin.

The purpose of this simple form validation is to check custom selectbox jquery plugin.

Custom selectbox Features: show

  • 1. Up & Down key works to select list item.
  • 2. Enter works to select the item from list.
  • 3. Mouse click works to select the list item.
  • 4. Scrollbar hide, but you can scroll the selectbox list items later (working on it).
